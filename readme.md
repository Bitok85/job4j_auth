## job4j_auth

## Простой проект с демонстрацией работы REST:
- Архитектура
- Валидация моделей и разные способы обработки исключений.
- Элементарная работа с DTO.
- Применение ResponseEntity с целью практики на случай работы с legacy REST кодом содержащим @Controller вместо @RestController.

## Используемые технологии:
- Java17
- Spring boot 2.5.2
- Hibernate 5.6.11
- ThymeLeaf 2.5.2
- Liquibase 3.6.2
- Lombok 1.18.22
- BootStrap 5.2.2
- PostgreSQL-14
- H2 DB 2.1.212
- junit.jupiter 5.9.0
- AssertJ 3.23.1
- log4j 1.2.17


## Для корректной работы приложения необходимо установить:
- Java 17
- Apache Maven 3.x+
- PostgreSQL 14.

## Запуск приложения
- Создайте базу данных "cars" использоваием командной строки или в любой платформе для администрирования PostgreSQL, например PgAdmin.
- Запуск приложения производится из корневого каталога проекта с использованием Maven. Необходимо выполнить следующие команды:
  ```shell
  mvn spring-boot:run
  ```
